## 🚀 How to Use

### For Developers

1. **Use component variants**:
```tsx
<Button variant="default" size="lg">Apply Now</Button>
<Badge variant="success">Approved</Badge>
<Card>...</Card>
```

2. **Use color tokens**:
```
primary-600, success-600, warning-600, error-600
```

3. **Use spacing scale**:
```
p-4, gap-6, m-8 (use predefined values)
```
